# Практическая работа №1 
## Программа калькулятор для посчета суммы которую заплатит каждый из гостей.
---
### Класс Main.
Данный класс является "мозгом" программы.
Он запускает следующие методы:
1. welcome(); - Запускает приветствие пользователя.
2. guest(); - Запускает цикл запроса колличества гостей. Ожидает от пользователя корректное число больше 1.
3. costOfProducts(); - Запускает цикл по запросу товара и его цены. По вводу "Завершить" выводит весь список товаров и общую стоимость.
4. finalCost(totalCost); - Запускает метод по подсчету суммы которую заплатит каждый из гостей.
5. goodBye(nameUser); - Запускает прощание с пользователем.
6. P.s И маленькая пасхалочка от меня:)
---
### Класс Calculator.
Данный класс выполняет подсчеты по колличеству гостей, и суммы которую заплатит каждый из гостей.

---
### Класс Products.
Данный класс запрашивает у пользователя товары и их цену с помощью ArrayList. И выводит общую стоимость всех введенных товаров.
